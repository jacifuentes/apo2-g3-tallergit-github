# apo2-g3-tallergit-github
Taller Git y GitHub para APO2  
Este repositorio es base para las asignaciones del taller Git y GitHub para el curso de Algoritmos y Programación 2. Una vez aceptadas las asignaciones, se crearán forks de la copia. Si esta copia se modifica y se desea actualizar en los repos de los estudiantes, se puede ejecutar la opción "Sync assignments" en el tablero de asignaciones, para abrir Pull Requests en todos los repositorios de los estudiantes.
